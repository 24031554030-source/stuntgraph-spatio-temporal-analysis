# 🧒 StuntGraph: Graph-Based Spatio-Temporal Analysis of Stunting in Indonesia

Graph-Based Spatio-Temporal Analysis of Socioeconomic Factors Affecting Stunting Prevalence Across Indonesian Provinces Using PCA, Multiple Linear Regression, Spatial Network Construction, and Community Detection.
## 📌 Project Overview

StuntGraph is a data mining project that investigates the socioeconomic factors influencing stunting prevalence across Indonesian provinces. By integrating statistical analysis, spatial network construction, and graph mining techniques, the project uncovers hidden relationships between provinces and identifies communities with similar stunting characteristics to support evidence-based public health policies.
## 🎯 Objectives

- Identify key socioeconomic factors associated with stunting prevalence.
- Reduce feature dimensionality using Principal Component Analysis (PCA).
- Model the relationship between socioeconomic variables and stunting prevalence using Multiple Linear Regression.
- Construct a spatial similarity network between provinces.
- Detect provincial communities using graph-based community detection algorithms.
## 📊 Dataset

The analysis utilizes publicly available Indonesian provincial socioeconomic and health indicators, including:

- Stunting prevalence
- Poverty rate
- Education indicators
- Healthcare accessibility
- Sanitation indicators
- Population statistics

The integrated dataset represents all Indonesian provinces during the study period.
